# Laravel Design Drawer

Slide-out drawer/panel component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/drawer
```

## Usage

### Livewire Component

```blade
<livewire:ld-drawer />
```

### Blade Component

```blade
<x-ld-drawer />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-drawer-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-drawer-views
```
